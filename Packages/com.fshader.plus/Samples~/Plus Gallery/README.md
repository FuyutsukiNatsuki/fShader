# fShader Plus Starter Materials

Plus Water、Ice、GlassのBalanced Materialと簡易Gallery Sceneです。Coreの`Gallery Textures` Sampleを追加Importすると、Water Wave/Foam、Ice Frost/Crack、Glass Condensation用Mapも利用できます。

Screen RefractionとLTCGIは安全のためOFFです。LTCGIを試す場合は公式Controller/Screen/Emitterを設定し、Affected Renderersを更新してからMaterial側をONにしてください。
