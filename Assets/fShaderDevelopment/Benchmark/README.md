# fShader Development Benchmark

This folder is isolated from the default VRChat scene.

The P0 scene establishes stable hierarchy names for later benchmark content:

- P0_Setup
- P0_Water_25pct
- P0_Ice_25pct
- P0_Glass_1_Layer
- P0_Glass_2_Layers

Do not add a default-on VRChat mirror. Mirror comparison is introduced as an
explicit, disabled benchmark case during the VRChat QA phase.
